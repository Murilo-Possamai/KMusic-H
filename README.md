## Sobre o Projeto

**KMusic/H** é um aplicativo mobile desenvolvido em Flutter que adapta automaticamente a sua playlist de acordo com a velocidade em que você está dirigindo. Dirija com prudência e deixe a trilha sonora acompanhar o seu trajeto.

> Projeto desenvolvido para a disciplina de **Dispositivos Móveis** do curso de **Análise e Desenvolvimento de Sistemas** no **Instituto Federal de Santa Catarina (IFSC)**.

---

## Funcionalidades

- **Autenticação** — Login e cadastro de usuário via Firebase Auth
- **Player de Música** — Tela principal com controles de reprodução
- **Playlists** — Gerenciamento e listagem de playlists
- **Velocímetro Inteligente** — Detecta a velocidade do usuário e adapta a música em tempo real
- **Suporte Android** — Desenvolvido com Flutter para Android

## Como Executar

### Pré-requisitos

- [Flutter SDK](https://docs.flutter.dev/get-started/install) `^3.x`
- [Firebase CLI](https://firebase.google.com/docs/cli) configurado
- Emulador Android/iOS ou dispositivo físico

### Instalação

```bash
# Clone o repositório
git clone https://github.com/Murilo-Possamai/KMusic-H.git

# Acesse a pasta do projeto
cd KMusic-H/kmusich

# Instale as dependências
flutter pub get

# Execute o aplicativo
flutter run
```
